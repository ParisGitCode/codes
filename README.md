# My Clicker Game

I made a simple clicker game with a score system and a emojis, the game objetive is toggling through the different emojis.
- GitHub Repository: https://github.com/ParisGitCode/codes

# My Project Description
My project is an engaging clicker game designed to be a visual experience rather than a step-by-step game. The core mechanic revolves around clicking the emoji at the centre to increase the ‘score’, and change the expression. 

The main themes I implemente for this game were loops and user interface (pretty much only through the mouse rather than keyboard)
I added user feedback loops and aim to add  sound effects upon click, and score increments to create a sense of reward. The scoring or point system is entirely there for positive feedback
- Aesthetic user interface which I used solely because I wanted more of a retro feel, something that isn’t too in-your-face like many modern games.

My technical approach was solely
P5.js, and FireAlpaca for drawings and graphics.

I worked on my own for this project, and split everything into chunks.
At first I focused on the click logic and score system, soon after I changed my mind since I didn’t want it to be an actual game with scores and points and a progress system, but I did like the scoring so I kept it (just for satisfaction, I like points!) after that, I focused on the music, this was the hard part.
I’m not too experienced in making music so I tried a couple notes on Bandlab, the aim was to make it more of a soft Lofi type of song but it ended up sounding a lot more electronic which wasn’t really what I was going for but I got a little carried away, also I wasn’t really able to mix as well as I’d like to.
I designing and drew all the custom character graphics, I initially wanted them to be really detailed, like animated characters but ultimately it was rather out of place for the type of project.

My sources of inspiration were:

- There's a game called Clicker Heroes on Cool Math Games that I used to play, which was really my first introduction to these type of game. It's a little unheard of but it definitely was something I had in mind while making this.

- Similarly, another website I had seen some time ago that my sister had shown me, it was a GitHub project and it was like a blob of slime, that would sort of indent and respond to the mouse's motion. I can't remember the name of it (or if it's even still up) but it was incredibly fascinating to me.
